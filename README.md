# Personal Portfolio Website

A clean, responsive, and modern personal portfolio website built with HTML, CSS, and JavaScript.

1.  **Personal Details**: Open `index.html` and replace the placeholders (e.g., "Name", "Tech Corp", "Project One") with your actual information.
2.  **Images**: 
    - Replace the profile picture placeholder in `index.html` with your own image path.
    - You can add images to an `images/` folder if you create one.
3.  **Links**: Update the `href` attributes in the social links section with your actual profile URLs.
4.  **Colors**: Open `style.css` and modify the `:root` variables to change the color scheme.

## ðŸš€ Deployment to GitHub Pages

1.  **Create a Repository**: Go to GitHub and create a new public repository (e.g., `my-portfolio`).
2.  **Push Code**:
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/my-portfolio.git
    git push -u origin main
    ```
3.  **Enable GitHub Pages**:
    - Go to your repository **Settings**.
    - Click on **Pages** in the left sidebar.
    - Under **Build and deployment**, select **Source** as `Deploy from a branch`.
    - Select `main` branch and `/ (root)` folder.
    - Click **Save**.

Your site will be live at `https://YOUR_USERNAME.github.io/my-portfolio/`.
